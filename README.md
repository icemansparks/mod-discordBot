# AzerothCore Module: Discord Bot Notifier

## Description
This module sends event-based messages from AzerothCore to a Discord channel using webhooks.

## Installation
1. Place `mod-discordBot` in your AzerothCore `modules/` directory.
2. Run `cmake` and `make` to compile the module.
3. Configure `conf/mod-discordBot.conf` and restart your server.

## Configuration
Set your Discord webhook URL in the `mod-discordBot.conf` file.
